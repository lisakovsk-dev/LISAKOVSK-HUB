# Инструкция по генерации отчетов

1. Используй предоставленные данные для формирования отчета.
2. Структура отчета должна включать: Заголовок, Введение, Основную часть, Выводы и Рекомендации.
3. Используй Markdown для форматирования (заголовки, списки, жирный шрифт).
4. Убедись, что тон отчета профессиональный и лаконичный.
5. Проверь отчет на отсутствие орфографических и логических ошибок.
